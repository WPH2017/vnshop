# 全栈电商项目VNShop
> 基于静态项目[shop-static] 构建Vue项目[vnshop]

## 文件结构

```bash
# 创建文件结构如下：
# ├vnshop
# ├─client # 前端
# ├─server # 后端
# ├─docs   # 文档
cd vnshop
mkdir -p client server docs

# client
# 1. 下载静态项目
git clone https://github.com/itguide/shop-static.git

# 2. 在同级目录使用webpack模板client，配置过程一路回车
vue init webpack client 

# 3. 安装依赖然后测试初始化成功
cd client
cnpm i
npm run dev
```
